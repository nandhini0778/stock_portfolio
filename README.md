# Stock Market Portfolio Tracker

A simple desktop application built using Python and Tkinter that allows users to manage and track their stock investments.

## Overview

This project was developed to demonstrate basic financial data management using a graphical user interface.  
The application allows users to enter stock details and calculate the total investment value of their portfolio.

It provides a simple way to record and view stock holdings without using complex financial platforms.

## Features

- Add stock name
- Enter number of shares
- Record purchase price
- Calculate total investment value
- Simple GUI built using Tkinter

## Technologies Used

- Python
- Tkinter (GUI library)
- Basic Python data structures

## How the Application Works

1. The user enters stock details:
   - Stock name
   - Quantity of shares
   - Purchase price

2. The application stores the entered information.

3. The program calculates:
   - Total investment value
   - Portfolio summary

4. The results are displayed in the GUI window.
